# bat文件

## 1. 命令

- **echo**

  ```c#
  // 打印 hello world
  echo hello world
  // 关闭回显
  @echo off
  ```

- **call**

  ```c#
  // 调用其他bat程序，而不终止原程序
  call ../xx.bat
  ```

- **pause**

  ```c#
  请按任意键继续，用于终止或暂停程序
  ```

- **rem** 或 **::**

  ```c#
  bat文件注释，类c#的//
  ```

- **set：**定义变量

  ```c#
  // 定义xxx变量并赋值
  set xxx = "111" 
  
  // set /p 设置变量，值由用户输入
  set /p xxx = 用户输入 
  
  // %xxx%表示使用变量
  dir /w %xxx% 
  
  // 删除xxx变量
  set xxx =  
  ```

- choice

  ```c#
  
  ```

- **exit**

  ```c#
  结束程序
  ```

  

- **%~dp0：**获取当前路径

  ```c#
  // 进入批处理所在目录
  cd %~dp0
  // 进入批处理所在目录的bin目录
  cd %~dp0bin\
  ```
